# Gate to Heaven Microsite

This directory contains the standalone microsite HTML files for "Gate to Heaven" / "Tor zum Himmel".

## Structure

- **Source files** (edited here):
  - `/src/de/gate-to-heaven/site.html` - German version
  - `/src/en/gate-to-heaven/site.html` - English version

- **Built files** (generated by Eleventy):
  - `/de/gate-to-heaven/site/` - German microsite (accessible at `/de/gate-to-heaven/site`)
  - `/en/gate-to-heaven/site/` - English microsite (accessible at `/en/gate-to-heaven/site`)

## URLs

- German: https://www.veithelmer.com/de/gate-to-heaven/site
- English: https://www.veithelmer.com/en/gate-to-heaven/site

## Assets

All film assets (posters, images, trailer) are copied from `content/films/gate-to-heaven/` to `_site/assets/films/gate-to-heaven/` during the build process.

## Development

To make changes:
1. Edit the HTML files in `/src/de/gate-to-heaven/site.html` or `/src/en/gate-to-heaven/site.html`
2. Run `npm run build` or `npm run dev` to rebuild
3. Check the output in `_site/de/gate-to-heaven/site/` or `_site/en/gate-to-heaven/site/`

## Design

The microsite recreates the look and feel of the original Flash-based website with:
- Blue color scheme (#0099CC) matching the original
- Four-column layout for content sections (Story, Cast, Trailer, Photos)
- Responsive design for mobile devices
- Embedded video player for the trailer
- Interactive photo gallery
